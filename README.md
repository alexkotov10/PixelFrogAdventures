# PixelFrogAdventures

A 2D Platformer game using Unity, with many mechanics including enemies, animations, projectiles, collectibles and more.

![Live Demo](https://alexkotov10.github.io/PixelFrogAdventures/)

## Controls

ASD - Movement

R - Reset

Spacebar - Jump
